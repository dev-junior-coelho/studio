🚀 COMO USAR (SUPER RÁPIDO):

  ─────────────────

  Procure por:
    enabled: false
  
  Mude para:
    enabled: true
  
  ✅ Pronto! O popup aparecerá para todos!


  DESATIVAR MANUTENÇÃO:
  ────────────────────
  Abra: src/lib/maintenance.ts
  
  Procure por:
    enabled: true
  
  Mude para:
    enabled: false
  
  ✅ Pronto! O login volta ao normal!

═══════════════════════════════════════════════════════════════════════════════

🔐 O QUE ACONTECE DURANTE A MANUTENÇÃO:

  1️⃣  Usuário acessa o app
      └─ Vê o popup de manutenção impossível de fechar

  2️⃣  Usuário tenta fazer login
      └─ Recebe erro: "Sistema está em manutenção"

  3️⃣  Dados de autenticação são limpos
      └─ localStorage e sessionStorage são zerados

  4️⃣  Nenhum usuário consegue acessar o sistema
      └─ Mesmo usuários já logados são desconectados ao recarregar
